## Installation

### CLI

```package-exec
solidui-cli@latest add menubar
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/menubar.tsx" showLineNumbers

```
