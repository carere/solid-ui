## Installation

### CLI

```package-exec
solidui-cli@latest add tooltip
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/tooltip.tsx" showLineNumbers

```

## Usage

```tsx
import { Tooltip, TooltipContent, TooltipTrigger } from "~/components/ui/tooltip";
```

```tsx showLineNumbers
<Tooltip>
  <TooltipTrigger>Hover</TooltipTrigger>
  <TooltipContent>
    <p>Add to library</p>
  </TooltipContent>
</Tooltip>
```
