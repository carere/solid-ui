## Installation

### CLI

```package-exec
solidui-cli@latest add card
```

### Manual

Copy and paste the following code into your project.

```file="~/registry/ui/card.tsx" showLineNumbers

```

## Usage

```tsx
import {
  Card,
  CardAction,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle
} from "~/components/ui/card";
```

```tsx
<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card Content</p>
  </CardContent>
  <CardFooter>
    <p>Card Footer</p>
  </CardFooter>
</Card>
```

## API Reference

### Card

The main container for the card component.

| Prop   | Type                   | Default     |
| ------ | ---------------------- | ----------- |
| `size` | `"default" \| "sm"`    | `"default"` |

### CardHeader

A container for the card's header content. Supports grid layout with `CardAction`.

### CardTitle

The title element of the card.

### CardDescription

The description element of the card.

### CardAction

An action element positioned in the top-right of the header.

### CardContent

The main content area of the card.

### CardFooter

The footer area of the card.
