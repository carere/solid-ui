## Installation

### CLI

```package-exec
solidui-cli@latest add badge
```

### Manual

Copy and paste the following code into your project.

```file="~/registry/ui/badge.tsx" showLineNumbers

```

## Usage

```tsx
import { Badge } from "~/components/ui/badge";
```

```tsx
<Badge variant="default">Badge</Badge>
```
