## Installation

### CLI

```package-exec
solidui-cli@latest add input
```

### Manual

Copy and paste the following code into your project.

```file="~/registry/ui/input.tsx" showLineNumbers

```

## Usage

```tsx
import { Input } from "~/components/ui/input"
```

```tsx showLineNumbers
<Input />
```
