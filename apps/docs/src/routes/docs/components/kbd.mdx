## Installation

### CLI

```package-exec
solidui-cli@latest add kbd
```

### Manual

Copy and paste the following code into your project.

```file="~/registry/ui/kbd.tsx" showLineNumbers

```

## Usage

```tsx
import { Kbd, KbdGroup } from "~/components/ui/kbd";
```

```tsx
<Kbd>⌘</Kbd>
```

## Examples

### Basic Keys

Display individual or grouped keyboard symbols:

```tsx
<KbdGroup>
  <Kbd>⌘</Kbd>
  <Kbd>⇧</Kbd>
  <Kbd>⌥</Kbd>
  <Kbd>⌃</Kbd>
</KbdGroup>
```

### Key Combinations

Show keyboard shortcuts with separators:

```tsx
<KbdGroup>
  <Kbd>Ctrl</Kbd>
  <span>+</span>
  <Kbd>B</Kbd>
</KbdGroup>
```

## API Reference

### Kbd

The `Kbd` component displays a single keyboard key.

| Prop    | Type                    | Default |
| ------- | ----------------------- | ------- |
| `class` | `string \| undefined`   |         |

### KbdGroup

The `KbdGroup` component groups multiple keyboard keys together.

| Prop    | Type                    | Default |
| ------- | ----------------------- | ------- |
| `class` | `string \| undefined`   |         |
